Welcome to the installation section. This documentation provides two distinct guides to cater to different needs:

  1. Docker Guide: For the fastest way to set up ErbslandFORMER on a server that supports Docker containers, refer to the docker installation guide.

  2. Step-by-Step Guide: A comprehensive step-by-step guide that offers detailed instructions for each step of the installation process. This guide is ideal for users who are setting up the application for the first time and need thorough explanations.

  3. System Administrators Guide: A condensed guide for system administrators that contains only the essential setup information in a compact format. This guide is suitable for experienced system administrators who are familiar with the general installation procedures and need a quick reference.
