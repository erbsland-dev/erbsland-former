============
Installation
============

Welcome to the installation section. This documentation provides two distinct guides to cater to different needs:

1. **Step-by-Step Guide**: A comprehensive :ref:`step-by-step guide<step-by-step-installation>` that offers detailed instructions for each step of the installation process. This guide is ideal for users who are setting up the application for the first time and need thorough explanations.

2. **System Administrators Guide**: A condensed :ref:`guide for system administrators<sysadmin-installation>` that contains only the essential setup information in a compact format. This guide is suitable for experienced system administrators who are familiar with the general installation procedures and need a quick reference.
